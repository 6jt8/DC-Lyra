<h2 align="center">✨ Features</h2>

<table align="center" width="100%">
  <tr>
    <td width="50%" valign="top" align="center" style="padding: 15px;">
      <h3>🎵 Music Playback</h3>
      <ul>
        <li>High-quality audio via Lavalink</li>
        <li>YouTube, Spotify, SoundCloud support</li>
        <li>Queue management (add, remove, skip, jump, move)</li>
        <li>24/7 mode in voice channels</li>
        <li>Autoplay & vote-skip system</li>
        <li>Volume control, seek, shuffle</li>
        <li>Audio filters (bassboost, nightcore, 3D, etc.)</li>
      </ul>
    </td>
    <td width="50%" valign="top" align="center" style="padding: 15px;">
      <h3>🎨 Player Experience</h3>
      <ul>
        <li>Live now-playing panel with buttons</li>
        <li>Synced lyrics display</li>
        <li>Generated song cards (Canvas)</li>
        <li>Progress bar & visualizer</li>
        <li>Multilingual UI (7 languages)</li>
        <li>Custom emoji system (auto-upload)</li>
        <li>Responsive status & activity</li>
      </ul>
    </td>
  </tr>
</table>
</div>

<h2 align="center">🚀 Quick Start</h2>

<table align="center" border="0">
<tr>
<td width="33%" align="center" valign="top">

<details>
<summary><b>🍞 Bun (Source)</b></summary>

```bash
unzip lyra-music-*-src.zip
cd lyra-music
bun install
cp .env.example .env
bun run start:bun
```

</details>

</td>
<td width="33%" align="center" valign="top">

<details>
<summary><b>🟢 Node.js (Dist)</b></summary>

```bash
unzip lyra-music-*-dist.zip
cd lyra-music
npm install
cp .env.example .env
npm start
```

</details>

</td>
<td width="33%" align="center" valign="top">

<details>
<summary><b>📦 From Source</b></summary>

```bash
git clone https://github.com/sayrox106/Lyra-MC.git
cd Lyra-MC
bun install   # or: npm install
bun run build # or: npm run build
bun start     # or: npm start
```

</details>

</td>
</tr>
</table>
